Number Systems Reference Guide

Binary (Base 2):
Uses digits: 0, 1
Position values: ...16, 8, 4, 2, 1
Example: 1010₂ = 1×8 + 0×4 + 1×2 + 0×1 = 10₁₀
Decimal (Base 10):
Uses digits: 0-9
Position values: ...1000, 100, 10, 1
Example: 123₁₀ = 1×100 + 2×10 + 3×1
Hexadecimal (Base 16):
Uses: 0-9, A-F (A=10, B=11, C=12, D=13, E=14, F=15)
Position values: ...256, 16, 1
Example: 1A₁₆ = 1×16 + 10×1 = 26₁₀
💡 Quick Tips:
  • Binary: Count in powers of 2
  • Hex: Remember A=10, B=11, C=12, D=13, E=14, F=15
  • Convert step by step - break down complex numbers
